******** Nibiru Studio Demo Source Code ********

* As an Android Studio module, developer should import it in an existed Android Studio project.

* Android Studio version for Demo is 3.0.1.

* Niburu Studio only support running on HMD devices with Nibiru OS. General Android System cannot provide required VR/AR features. If use mobile to debug, install NibiruXRMoibleLauncher first and then start your app in launcher, but the App will lose all optimiziation and features in this way. Nibiru don't make sure it can run on all Non-Nibiru OS devices.




******* Nibiru Studio Demo源码 **********

* Nibiru Studio Demo源码是一个Android Studio的Module，因此不能直接打开，而是应导入到一个已存在的Android Studio项目中（Project）

* Android Studio开发Demo的版本为3.0.1

* Nibiru Studio只支持在Nibiru系统上运行，因为一般的Android系统无法提供完整的VR/AR支持。如果需要使用手机调试，首先安装NibiruXRMoibleLauncher，然后在launcher中启动应用。Nibiru并不能确保Nibiru Studio开发的应用在所有非Nibiru系统上正常运行。
